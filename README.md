# BlockChain-Viewer
* This is the repo for BlockChain Course Final Project
* This Repo consists of both front-end and backend
* Technology used in this project
    * Frontend
        * Angular Framework
        * CoreUI
        * Typescript
    * Backend
        * NodeJS
        * Express Framework
        * Modules
            * Auth Module
                * Login
                * Register
            * ViewEvent Module
                * View Food Content
                * View Food Image
                * View Food Image Replace
                * View Food Section
***
## How to start Client
```
cd DashBoard-Client
npm install // for the first time
ng serve // open browser, type localhost:4200 open the client
```
## How to start Server
```
cd Server
npm install // for the first time
npm start // open port on 8000
```
